# Weather Forecast Application

A beautiful and simple Full-Stack Weather Application built using **Java (Spring Boot)** for the backend and **HTML/CSS/JavaScript** for the frontend. It fetches live 5-day weather data using the OpenWeatherMap API.

## 🚀 Features
* **Live Weather Data:** Fetches real-time temperature and humidity.
* **5-Day Forecast:** Displays daily weather forecast for 12:00 PM blocks.
* **Location Selection:** Search weather for any city globally (e.g., Delhi, Mumbai, London).
* **Responsive UI:** Clean card layout with a gradient background.

## 🛠️ Tech Stack
* **Backend:** Java 21, Spring Boot 3.x, Spring Web (REST API), RestTemplate
* **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
* **External API:** OpenWeatherMap API

## 📸 How to Run Locally
1. Clone this repository.
2. Open the project in **Eclipse IDE** as an *Existing Maven Project*.
3. Open `src/main/java/com/example/weatherapp/WeatherService.java` and add your **OpenWeatherMap API Key**.
4. Right-click `WeatherappApplication.java` -> **Run As -> Java Application**.
5. Open your browser and go to: `http://localhost:8080/`
